    **Component 2: `src/components/agents/memory-file-editor.tsx`**

    Props:
    ```typescript
    interface MemoryFileEditorProps {
      agentId: string
      filePath: string | null
      isViewer: boolean
      isAgentRunning: boolean
      onDirtyChange?: (dirty: boolean) => void
    }
    ```

    Implementation:
    - Fetches file content from `GET /api/agents/{agentId}/files/{filePath}` when filePath changes. Shows loading spinner while fetching.
    - State: `content` (current editor text), `originalContent` (from API), `mode` ('preview' | 'edit'), `isDirty` (content !== originalContent), `readOnly` (from API response), `saving` (loading state for save).
    - When filePath is null, show empty state: "Select a file to view" centered message.
    - When file loaded and readOnly is true, show lock icon + "Auto-generated by agent" label below the filename. Do NOT show Edit button.

    **Header bar** (above the content area):
    - Filename displayed as `text-sm font-medium`. If isDirty, append a dot indicator (small colored circle, `bg-warning` ~6px).
    - Mode toggle buttons: "Preview" and "Edit" -- styled as small pill buttons. "Edit" button hidden when `isViewer` or `readOnly`.
    - "Save" button (right-aligned): visible only in edit mode, disabled when !isDirty or saving. Styled with `bg-primary text-primary-foreground` when active.

    **Agent running warning** (below header, above content):
    - If `isAgentRunning` AND mode === 'edit', show yellow banner: `bg-warning/10 border border-warning/30 text-warning text-xs p-2 rounded-md` with text "Agent is running -- changes may be overwritten by the agent".

    **Content area:**
    - Preview mode: Render with `<MarkdownRenderer content={content} />`. Wrap in a scrollable container.
    - Edit mode: `<textarea>` that fills the available space. Use `className="w-full h-full min-h-[200px] bg-transparent text-sm text-foreground font-mono resize-none p-2 focus:outline-none"`. Bind to content state, update isDirty on change.

    **Save handler:**
    - `PUT /api/agents/{agentId}/files/{filePath}` with `{ content }` body.
    - On success: update originalContent, set isDirty false, show toast `{ message: 'Saved', type: 'success' }`.
    - On error: show toast `{ message: 'Failed to save', type: 'error' }`.

    **Ctrl+S handler:**
    - useEffect with keydown listener for `(e.metaKey || e.ctrlKey) && e.key === 's'`. Prevent default. Call save if isDirty and not isViewer and not readOnly.

    **Dirty change callback:**
    - Call `onDirtyChange?.(isDirty)` whenever isDirty changes, so the parent (MemoryTab) can implement the unsaved changes guard.
